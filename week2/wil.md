# 2주차 WIL 2026/04/01

이번 스터디에서는 지도학습과 비지도학습의 개념을 배우고,
KNN 알고리즘을 이용한 분류 실습을 진행하였다.

특히 훈련 세트와 테스트 세트를 나누지 않고 모델을 평가하면
정확도 100%가 나와도 의미가 없다는 점을 알게 되었고,
이를 통해 데이터 분리의 중요성을 이해하게 되었다.

또한 데이터를 잘못 나누면 특정 데이터만 학습하게 되는
샘플링 편향 문제가 발생할 수 있다는 것도 확인하였다.

이후 train_test_split을 이용하여 데이터를 적절히 섞고 나누는 방법을 학습하였고,
데이터의 비율을 다르게 설정하면서 모델의 성능 변화를 확인하였다.

특히 길이와 무게처럼 스케일이 다른 데이터에서는
거리 기반 알고리즘인 KNN이 제대로 동작하지 않는 문제가 발생하였다.

이를 해결하기 위해 평균과 표준편차를 이용한 표준점수 방식으로
데이터를 스케일링하였고, numpy를 사용하지 않고 직접 평균, 분산,
표준편차를 계산해보며 전처리 과정을 이해할 수 있었다.

이번 실습을 통해 머신러닝에서는 모델 자체뿐만 아니라
데이터 전처리가 매우 중요하다는 것을 느끼게 되었다.
