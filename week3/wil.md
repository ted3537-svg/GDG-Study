# 3주차 WIL 2026/04/09

이번 AI 스터디 3주차에서는 회귀 모델과 규제에 대해 학습하고, 이를 직접 코드로 구현해보았다.

먼저 KNN 회귀, 선형 회귀, 다항 회귀를 통해 다양한 회귀 모델의 동작 방식을 이해하였다. 특히 PolynomialFeatures를 사용하여 degree를 증가시키면 특성의 개수가 늘어나면서 모델이 더 복잡해진다는 것을 확인하였다.

실습에서는 degree=15로 설정하여 다중 회귀 모델을 학습해보았는데, train score는 매우 높게 나오지만 test score는 상대적으로 낮게 나오는 과대적합 현상을 직접 확인할 수 있었다. 이를 통해 모델이 지나치게 복잡해지면 데이터의 패턴을 학습하는 것이 아니라 데이터를 외워버리는 문제가 발생한다는 것을 이해하게 되었다.

이 문제를 해결하기 위해 StandardScaler를 사용하여 데이터를 스케일링하였다. 거리 기반 또는 계수 기반 모델에서는 각 특성의 값 범위가 다르면 학습에 영향을 주기 때문에, 평균과 표준편차를 이용해 데이터를 정규화하는 과정이 필요하다는 것을 배웠다.

이후 Ridge와 Lasso 규제를 적용하였다. Ridge는 모든 계수를 작게 만들어 모델의 복잡도를 줄이고, Lasso는 일부 계수를 0으로 만들어 불필요한 특성을 제거하는 방식으로 작동한다는 것을 코드로 확인하였다. 수업에서 말한 모든 문제집을 다 보되, 일회독만 하며, 깊게
공부하지 않도록 제한한다는 말과 필요 없는 문제집은 아예 제외하고 중요한
것만 집중해서 공부하는 방식이란 말이 잘 와닿았다.

규제를 적용한 결과 train score와 test score의 차이가 줄어들었고, 모델이 과대적합 상태에서 벗어나 보다 일반화된 성능을 가지게 되는 것을 확인할 수 있었다.

이번 과제를 통해 단순히 모델을 만드는 것보다, 데이터 전처리와 모델의 복잡도를 조절하는 것이 더 중요할 수 있다는 것을 느꼈으며, 특히 과적합을 방지하기 위한 스케일링과 규제의 필요성을 이해할 수 있었다. 코딩 실력이 아직 부족하여 혼자 AI 코드를 구현하는 것은 힘들지만, 수업을 통해 AI의 작동 방식을 이해하는 것이 흥미로웠다.
